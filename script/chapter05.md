# Глава 5. Ночь чужих гаражей

**Персонажи**
- **Олег**, **Женек**, **Вадим**, **Тёмка**
- **Чужие «мы»** — версии компании в чужих боксах
- **Жёлтые глаза** — то, что ползёт между гаражами (отсылка к «Рикки-Тикки-Тави»)

**Стиль:** ч/б треш-комикс + психоделический new wave из клипов Женька.

**Связь с гл. 4:** рука из белого ТВ сжимает в ответ. Их вытаскивает **наружу** — но не на улицу города, а в ночь бесконечной аллеи. Дверь 236 за ними захлопывается. Выбор «разбить экран» / «стук» тоже сюда сходятся: петля всё равно выплёвывает их на переулок чужих боксов.

**Отсылки к клипам Женька**
- [Аниме](https://www.youtube.com/watch?v=eZYFKmXdsqs): «улетаю в аниме», «хаджиме», жизнь «не для показа», жулик который ещё живой; визуал — CRT с человеком внутри, белый кирпич, кресло, силуэт на стене
- [Рикки-Тикки-Тави](https://www.youtube.com/watch?v=Kr2DMk3AESA): «горячий коридор», «я не боюсь», «мангус», жёлтые глаза из темноты, «дом мой дом»; визуал — белая рубашка + галстук, шрам, brick noir, коридор теней

## Заметки по продакшену

- **Старт:** кадр 01 обязан рифмоваться с финалом гл. 4 (рука / белый CRT) и с кадром CRT-в-CRT из клипа «Аниме».
- **Женек — проводник:** его реплики не «пасхалки ради пасхалок», а рабочий язык навигации по петле.
- **Визуальные якоря клипов:**  
  - 08 — CRT + кресло + белый кирпич («Аниме»)  
  - 09 — горячий коридор  
  - 10–11 — жёлтые глаза → рубашка/галстук/шрам («Рикки»)  
  - 20 — Другой-Женек в образе из клипа
- **Чужие гаражи (04–06, 13, 15):** каждый = одна идея, без перегруза.
- **Амбиент:** alley → unease → horror → haze (anime) → panic → cliffhanger.
